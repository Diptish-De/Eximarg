# EXIMARG – Redesign the Entire Onboarding Journey (Levels 1–6)

The landing page design is now approved and acts as the visual foundation of EXIMARG.

**DO NOT redesign the landing page.**

Instead, redesign the **authenticated onboarding experience (Levels 1–6)** while preserving the same typography, spacing, color palette, editorial layout, and premium aesthetic.

The onboarding should feel like **Chapter 2 onward of the same story**.

The user should never feel like they entered a government portal or a generic SaaS dashboard.

---

# Core Vision

Design this product as **"The Guided Journey to Becoming a Global Exporter."**

The user is not completing forms.

The user is progressing through chapters.

Each completed level should feel like unlocking a new milestone in their business.

Every screen should communicate:

* Progress
* Trust
* Confidence
* Achievement
* Simplicity

---

# Design Language

Keep the exact visual style established on the landing page.

* Warm white background
* Soft ivory surfaces
* Royal blue accents
* Editorial serif headings
* Clean modern sans typography
* Large whitespace
* Rounded cards
* Soft premium shadows
* Minimal gradients
* Calm luxury
* Apple + Stripe + Linear quality

Do NOT switch to enterprise dashboard styling.

Do NOT use Bootstrap layouts.

Do NOT create long scrolling forms.

---

# Instead of One Long Form

Break every level into beautiful cards.

Each card represents one logical step.

Cards should have:

* soft shadow
* rounded corners
* icon
* short explanation
* inputs
* completion indicator

# UX Principles

One primary action.

Everything else is secondary.

Never overwhelm the user.

Break complex tasks into beautiful sections.

Use progressive disclosure.

Always explain why information is required.

Show rewards before asking for work.

Celebrate every achievement.

---

# Quality Target

This should feel like the onboarding experience of a billion-dollar product.

Think:

Apple onboarding

Stripe Identity

Shopify setup

Linear onboarding

Arc Browser

Notion AI

Never government forms.

Never enterprise ERP.

Never Bootstrap.

The user should enjoy completing each level because the experience feels guided, rewarding, elegant, and premium.

Maintain complete visual consistency with the approved landing page while redesigning every onboarding level using a modern card-based information architecture.
